# **App Name**: ClarityEye

## Core Features:

- Profile Management: User login and profile creation for doctors, nurses, and health workers.
- Patient Data Entry: Input patient information, including name, age, symptoms, and medical history.
- Image Upload: Upload images of the eye using the smartphone camera for AI analysis.
- AI-Powered Triage: Use AI tool and clinical logic to triage the urgency of the case: Urgent, Refer Within Week, or Manage Locally.
- Referral Options: Display list of ophthalmologists nearby, with Google ratings, social media reviews, and clinic directions, utilizing Google Maps API.
- Referral Generation: Generate a PDF referral document that can be printed or sent via SMS/email.

## Style Guidelines:

- Primary color: A vibrant blue (#2E9AFE) to represent trust, clarity, and medical professionalism.
- Background color: A light blue-gray (#F0F8FF), offering a clean, clinical feel.
- Accent color: A muted purple (#A020F0) used for highlights, CTAs, and to bring attention to important details; analogous to blue while remaining distinct.
- Body text: 'PT Sans', a sans-serif font, should be used for body text. It's modern yet with warmth and personality.
- Headline font: 'Playfair', a modern serif font, will convey an elegant and high-end feel, in keeping with the app's crucial medical function.
- Use clear, medical-themed icons to represent symptoms and urgency levels.
- Prioritize a clean and intuitive layout to make information easily accessible.
